# Mendeleev (Typescript training)

## Install

Open a command line tool (cmd).

```
git clone https://github.com/jlg-formation/tsr-juillet-2021.git
cd tsr-juillet-2021
cd ..
cd front
npm i
npm start
```

Open a second command line tool.

```
cd tsr-juillet-2021
cd back
npm i
npm start
```

## Run

Open a web browser on `http://localhost:3000`.
